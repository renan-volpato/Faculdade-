## 1) O que é Programação Orientada a Objetos?
É um modelo de programação que organiza o código em torno de objetos para representar elementos do mundo real.
Esses objetos juntam dados (atributos) e ações (métodos).

## 2) Vantagens da POO em relação à programação estruturada
- alto reaproveitamento de código
- organização superior através da modularidade
- melhor separação de responsabilidade de cada parte do sistema
- facilidade de manutenção
- maior segurança na proteção dos dados da aplicação

## 3) O que é um objeto na POO?
Objeto é uma instância do programa que contem: atributos e métodos.
Atributos são informações (exemplo: nome, idade, nacionalidade).
Métodos são as ações (exemplo: depositar, sacar, consultar).

## 4) 4 fundamentos (pilares) da POO
Encapsulamento: protege e controla o acesso aos dados internos do objeto, permitindo modificações apenas por meio de métodos definidos.
Abstração: representação apenas dos aspectos essenciais, ocultando detalhes desnecessários.
Herança: permite que uma classe possa aproveitar características de outra, estabelecendo uma relação hierárquica.
Polimorfismo: permite que o mesmo método ou ação possa funcionar de formas diferentes dependendo do objeto.
